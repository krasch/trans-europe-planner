Datenschutz

Die Reiseplanung findet komplett im Browser statt. Keinerlei Reisedaten oder persönliche Daten von Nutzer*innen werden an einen Server gesendet, es werden auch nirgendwo solche Daten gespeichert.

Anonyme Nutzungsstatistiken werden gesammelt durch https://plausible.io/, dies sind: Browser, Gerätetyp (Desktop oder Mobile), Betriebssystem, Land, HTTP Referrer. Mehr Details gibt es unter https://plausible.io/data-policy#first-thing-first-what-we-collect-and-what-we-use-it-for

Data protection

Travel planning happens completely in the browser. Neither trip info nor any personal user data are being sent to a server, such data is also not stored anywhere.

Anonymous usage statistics are being collected through https://plausible.io/, these are: browser, device type (desktop or mobile), operating system, country, HTTP Referrer. A full list can be found at https://plausible.io/data-policy#first-thing-first-what-we-collect-and-what-we-use-it-for
